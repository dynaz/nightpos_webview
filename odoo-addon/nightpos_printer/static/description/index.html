ข้อกำหนดระบบ

☁️ ฝั่ง Odoo (Cloud)

  • Odoo 19 (Community หรือ Enterprise)
  • Python 3.11+
  • Module web (installed by default)
  • addon nightpos_printer ติดตั้งแล้ว

📱 ฝั่ง Android

  • แอป NightPOS Android (Flutter) ติดตั้งแล้ว
  • Android 6.0+ (API 23)
  • อุปกรณ์ต้องอยู่ใน LAN เดียวกับพริ้นเตอร์

🖨️ พริ้นเตอร์

  • พริ้นเตอร์รองรับ ESC/POS protocol
  • เชื่อมต่อเครือข่าย (Ethernet หรือ Wi-Fi)
  • เปิด TCP Listener บน port 9100
  • ตัวอย่าง: Epson TM-T88, SEWOO, Bixolon, Star, Xprinter

🌐 เครือข่าย

  • Android กับ Printer ต้องอยู่ LAN เดียวกัน
  • ไม่จำเป็นว่า Odoo server จะต้องเข้าถึง printer
  • Firewall ต้องอนุญาต TCP port 9100

ตั้งค่าแอป Android

ทำไมต้องตั้งค่าทั้ง 2 ฝั่ง?
Odoo เก็บ IP/Port เพื่อให้ admin จัดการผ่าน backend แอป Android เก็บแยกเพื่อให้พิมพ์ได้แม้ออฟไลน์จาก Cloud เมื่อ Odoo เรียก JS handler จะส่ง IP/Port ไปตรงๆ โดยไม่ต้องพึ่ง Android registry

เปิดแอป NightPOS บนอุปกรณ์ Android

ไปที่ Settings → TCP Printers

กดไอคอน ⚙️ มุมบนขวา → เลื่อนลงหา TCP Printers → แตะ

เพิ่มพริ้นเตอร์

กด + เพิ่มพริ้นเตอร์ แล้วกรอก:

  • ชื่อพริ้นเตอร์ — ชื่อที่ต้องการแสดง
  • IP Address — เช่น 192.168.1.176
  • Port — 9100 (ค่าเริ่มต้น)
  • เปิด Default Printer ถ้าต้องการให้เป็นค่าเริ่มต้น

ทดสอบ (optional ณ จุดนี้)

กดปุ่ม Test ข้างชื่อพริ้นเตอร์เพื่อทดสอบ TCP connection ถ้าขึ้น "เชื่อมต่อสำเร็จ!" แสดงว่าพร้อมใช้งาน

JavaScript API

เรียกใช้ได้ 2 วิธี — ผ่าน Odoo service injection หรือ global window.NightPOSPrinter

JS isAvailable() → boolean

ตรวจสอบว่าเปิดอยู่ใน NightPOS Android app หรือไม่ ควรเรียกก่อนทุกครั้งก่อนพิมพ์

ตัวอย่าง
if (!window.NightPOSPrinter.isAvailable()) {
  alert('กรุณาเปิดผ่านแอป NightPOS เท่านั้น');
  return;
}
JS printRaw(base64Data, opts?) → Promise<{success, error?}>

ส่งข้อมูล ESC/POS ดิบ (encoded เป็น Base64) ไปพิมพ์โดยตรง

ParameterTypeRequiredคำอธิบาย
base64DatastringrequiredESC/POS bytes เข้ารหัส Base64
opts.ipstringoptionalIP ของพริ้นเตอร์เป้าหมาย
opts.portnumberoptionalTCP port (ค่าเริ่มต้น: 9100)
opts.printerIdstringoptionalAndroid printer UUID (แทน ip/port)
ตัวอย่าง — ส่ง ESC/POS base64 ตรงๆ
const escposB64 = 'G1hBAAAAAAA...'; // base64 ESC/POS data

// วิธีที่ 1: ระบุ IP ตรงๆ (Odoo ส่ง ip มาด้วย)
const res = await window.NightPOSPrinter.printRaw(escposB64, {
  ip: '192.168.1.176',
  port: 9100
});

// วิธีที่ 2: ใช้ default printer (ไม่ต้องระบุ opts)
const res = await window.NightPOSPrinter.printRaw(escposB64);

if (res.success) { console.log('พิมพ์สำเร็จ'); }
else             { console.error(res.error); }
JS printText(text, opts?) → Promise<{success, error?}>

ส่งข้อความ UTF-8 ธรรมดา — Android จะห่อด้วย ESC/POS envelope อัตโนมัติและตัดกระดาษให้

ตัวอย่าง
await window.NightPOSPrinter.printText(
  `================================
NightPOS Receipt
================================
Item A              100.00 THB
Item B               50.00 THB
--------------------------------
TOTAL               150.00 THB
================================
Thank you!
`,
  { ip: '192.168.1.176', port: 9100 }
);
JS printTextViaServer(text, printerId?) → Promise<{success, error?}>

Python server แปลงข้อความเป็น ESC/POS แล้วส่งกลับมา JS จึงค่อยพิมพ์ เหมาะเมื่อต้องการควบคุม formatting จาก Python

ตัวอย่าง — ให้ server generate ESC/POS
// printerId = Odoo nightpos.printer ID (null = default)
await window.NightPOSPrinter.printTextViaServer(
  receiptText,
  1  // Odoo printer ID
);
JS testConnection(ip, port?) → Promise<{reachable: boolean}>
ตัวอย่าง
const { reachable } = await window.NightPOSPrinter.testConnection('192.168.1.176');
if (!reachable) { alert('พริ้นเตอร์ไม่ตอบสนอง'); }

ดึงรายชื่อพริ้นเตอร์จาก Odoo database

ตัวอย่าง
const printers = await window.NightPOSPrinter.listOdooPrinters();

import { useService } from "@web/core/utils/hooks";

class MyComponent extends Component {

ตัวอย่าง A — ปุ่มพิมพ์ใบเสร็จใน Sale Order

PYTHON (models/sale_order.py)
from odoo import models

class SaleOrder(models.Model):
    _inherit = 'sale.order'

    def action_nightpos_print(self):
        self.ensure_one()

        # 1. สร้างข้อความใบเสร็จ
        lines = [f'{l.name[:20]:<20} {l.price_total:>8.2f}' for l in self.order_line]
        receipt = f"""
================================
{self.company_id.name}
================================
Order: {self.name}
Date : {self.date_order.strftime('%d/%m/%Y %H:%M')}
--------------------------------
{''.join(lines)}
--------------------------------
TOTAL: {self.amount_total:>18.2f} THB
================================
Thank you!
"""

        # 2. ดึง default printer แล้วสร้าง client action
        Printer = self.env['nightpos.printer']
        default = Printer.get_default_printer()
        if not default:
            raise UserError('ไม่มีพริ้นเตอร์ที่ตั้งค่าไว้')

        printer = Printer.browse(default['id'])
        return printer.get_print_action(text=receipt)

ตัวอย่าง B — พิมพ์อัตโนมัติจาก JavaScript เมื่อ POS ยืนยัน Order

JAVASCRIPT (POS หรือ Custom JS)
// ใช้ global window.NightPOSPrinter ที่ service register ไว้
async function printOrderReceipt(order) {
  if (!window.NightPOSPrinter?.isAvailable()) return;

  // ดึง printer จาก Odoo (เพื่อได้ ip:port)
  const printers = await window.NightPOSPrinter.listOdooPrinters();
  const def = printers.find(p => p.is_default) || printers[0];
  if (!def) { console.warn('No printer'); return; }

  // ให้ server generate ESC/POS แล้วพิมพ์
  await window.NightPOSPrinter.printTextViaServer(
    buildReceiptText(order),
    def.id
  );
}

function buildReceiptText(order) {
  return [
    `================================`,
    `Order: ${order.name}`,
    `================================`,
    ...order.lines.map(l => `${l.name.padEnd(20)} ${l.total.toFixed(2)}`),
    `--------------------------------`,
    `TOTAL: ${order.total.toFixed(2)} THB`,
    `================================\n`,
  ].join('\n');
}

ESC/POS Command Reference

คำสั่ง ESC/POS ที่ใช้ใน _generate_test_escpos() และ _text_to_escpos() สำหรับนักพัฒนาที่ต้องการ generate ESC/POS เองจาก Python

คำสั่ง (bytes)ความหมายPython
ESC @Initialize / Reset printerb'\x1b@'
ESC a 0จัดชิดซ้ายb'\x1ba\x00'
ESC a 1จัดกลางb'\x1ba\x01'
ESC a 2จัดชิดขวาb'\x1ba\x02'
ESC E 1ตัวหนา ONb'\x1bE\x01'
ESC E 0ตัวหนา OFFb'\x1bE\x00'
ESC ! 0x30Double width + heightb'\x1b!\x30'
ESC ! 0x00ขนาดปกติb'\x1b!\x00'
ESC d nFeed n บรรทัดb'\x1bd\x04'
GS V 0x41 0x00Partial cutb'\x1dV\x41\x00'
GS V 0x00Full cutb'\x1dV\x00'

ตัวอย่าง — Generate ESC/POS ใบเสร็จสวยงาม

PYTHON
def _build_receipt_escpos(self, order):
    ESC, GS = b'\x1b', b'\x1d'
    d = bytearray()

    # Header — ชื่อร้านกลาง ตัวใหญ่
    d += ESC + b'@'
    d += ESC + b'a\x01'          # Center
    d += ESC + b'E\x01'          # Bold
    d += ESC + b'!\x30'          # 2x size
    d += f"{order.company_id.name}\n".encode('utf-8')
    d += ESC + b'!\x00' + ESC + b'E\x00'

    # Order info
    d += ESC + b'a\x00'          # Left
    d += ('-' * 32 + '\n').encode()
    d += f"Order : {order.name}\n".encode('utf-8')
    d += f"Date  : {order.date_order:%d/%m/%Y %H:%M}\n".encode('utf-8')
    d += ('-' * 32 + '\n').encode()

    # Line items
    for line in order.order_line:
        name  = line.name[:20].ljust(20)
        price = f"{line.price_total:>9.2f}"
        d += f"{name}{price}\n".encode('utf-8')

    # Total
    d += ('=' * 32 + '\n').encode()
    d += ESC + b'E\x01'
    d += f"TOTAL{'':17}{order.amount_total:>9.2f} THB\n".encode('utf-8')
    d += ESC + b'E\x00'
    d += ('=' * 32 + '\n').encode()

    # Footer — ขอบคุณกลาง
    d += ESC + b'a\x01'
    d += 'ขอบคุณที่ใช้บริการ\n'.encode('utf-8')
    d += ESC + b'd\x04'           # Feed 4 lines
    d += GS  + b'V\x41\x00'      # Partial cut

    return bytes(d)

  • ต้องเปิด Odoo ผ่าน แอป NightPOS Android เท่านั้น ไม่ใช่ Chrome/Firefox
  • ตรวจสอบว่า addon nightpos_printer ติดตั้งสำเร็จ (เมนู NightPOS Printer ปรากฏ)
  • ลอง hard-refresh แอป (ปิดแอปแล้วเปิดใหม่)
ทดสอบ connection ล้มเหลว "เชื่อมต่อไม่ได้"
  • ตรวจสอบว่าพริ้นเตอร์เปิดอยู่และ Network LED ติด
  • ตรวจสอบ IP ด้วย ping 192.168.1.176 จาก Android (ใช้ Terminal app)
  • ยืนยันว่า Android อยู่ Wi-Fi เดียวกับพริ้นเตอร์ (ไม่ใช่ 4G/5G)
  • ตรวจสอบ Firewall ของ Router ว่าไม่บล็อก port 9100
  • ลองตรวจสอบ IP ของพริ้นเตอร์ด้วยการพิมพ์ Test Page จากหน้าจอพริ้นเตอร์โดยตรง
พิมพ์สำเร็จ แต่ตัวอักษรภาษาไทยแสดงผิดเพี้ยน/เป็นเครื่องหมายแปลก
  • พริ้นเตอร์อาจไม่รองรับ UTF-8 — ลองเปลี่ยนเป็น TIS-620: text.encode('tis-620', errors='replace')
  • บางรุ่นต้องส่งคำสั่ง select codepage ก่อน: ESC t n (n=21 สำหรับ TIS-620)
  • ลองใช้แค่ ASCII ก่อนเพื่อยืนยันว่าพิมพ์ได้
Module หาไม่เจอใน Apps list
  • ตรวจสอบ addons_path ใน odoo.conf ว่ามี D:\19odoo\custom
  • ต้อง restart Odoo service ก่อน Update Apps List
  • เปิด Developer Mode ก่อน (Settings → General Settings)
  • ดู log ที่ D:\19odoo\server\odoo.log หาข้อผิดพลาด
window.NightPOSPrinter เป็น undefined
  • addon ยังไม่ได้ติดตั้ง หรือ JS ยังไม่โหลด — ตรวจสอบ Network tab ใน DevTools
  • ลอง hard-refresh: Ctrl+Shift+R หรือเคลียร์ cache ใน Chrome Remote Debugging
  • ดู Console errors — อาจมี JS syntax error ในการ load asset
Error "No printer configured"
  • ยังไม่ได้เพิ่มพริ้นเตอร์ใน Odoo — ไปที่ NightPOS Printer → พริ้นเตอร์ → New
  • หรือยังไม่ได้ตั้ง Default — เปิดพริ้นเตอร์แล้วกด "ตั้งเป็น Default"
